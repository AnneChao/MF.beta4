MF.beta4 (R code and data for graphics in 2023 submission by Chao et al.)

R code and data are provided for plotting all figures in the submission "Hill-Chao numbers allow decomposing gamma-multifunctionality into alpha and beta components". 

"MF.beta4" includes the following files:

(1) Data Files: 

(1a) EFunctions data: "Ratcliffe_et_al_ELE_12849_ecosystem_function_variables.csv" (for figures 2, 4, 5 in the paper and appendices S3, S4). 

(1b) basal_area data: "FunDivEUROPE_BetaFor_species_composition.csv" (for Figures 2, 4, 5 in the paper and appendices S3, S4).

(2) R_code.R: Main code for plotting all figures (for figures 2, 4, 5 in the paper and appendices S3, S4).

(3) Source R code: "Source R code.txt".
